# Format Numbers

## Objective

Write a method that can take any integer and add commas. For instance:

| Argument | Return Value |
|----------|--------------|
| 35       | 35           |
| 1051     | 1,051        |
| 609124   | 609,124      |
| 23482243 | 23,482,243   |

## Instructions

This lab is test-driven so run `rspec` and get those tests to pass.
